; 본문: 선택사항 / 커밋에 설명이 필요한 경우 / 한 줄 72자 이내
; 꼬릿말 : 선택사항 / issue tracker ID를 명시하고 싶은 경우에 작성

; type
;   - Feat: 기능 추가
;   - Fix: 버그 수정
;   - Docs: 문서 수정
;   - Style: 코드 포맷 변경, 세미콜론 누락, 오타, 변수명 변경 등; 코드 수정이 없는 경우
;   - Refactor: 프로덕션 코드 리팩토링
;   - Test: 테스트 추가, 테스트 리팩토링; 프로덕션 코드 수정이 없는 경우
;   - Chore: 빌드 태스크 업데이트, 패키지 매니저 실행 등; 프로덕션 코드 수정이 없는 경우
; ========================
; footer
;   - Resolves: 이슈를 해결했을 때 사용
;   - See also: 참고할 이슈가 있을 때 사용
;   - Fixes: 이슈 수정중 (아직 해결되지 않은 경우)
;   - Related to: 해당 커밋에 관련된 이슈번호 (아직 해결되지 않은 경우)
; ========================
